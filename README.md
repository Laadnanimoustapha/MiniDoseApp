### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/MiniDoseApp.git
   ```

2. Navigate to the project directory:

   ```bash
   cd MiniDoseApp
   ```

3. Compile the program:

   ```bash
   g++ -o MiniDoseApp MiniDoseApp.cpp
   ```

4. Run the program:
